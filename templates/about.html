This App takes in stock Ticker Input from the user and plots the different prices.

The user can choose from one of the prices

▪ Opening Price.

▪ Adjusted Opening Price.

▪ Closing Price.

▪ Adjusted Closing Price.

▪ All the above prices.


The app plots one or all these prices for the chosen stock for the last month ending on the current date.
